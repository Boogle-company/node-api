# Boogle

Indexador de sites, muito divertido e animado.

## Como utilizar

Siga os passos abaixo para ter um melhor proveito da nossa plataforma

### Instalar Dependências

 ``` bash
 npm install
 ```

### Popular Banco de Dados

``` bash
npm run populate
```

### Rodar Servidor

``` bash
npm run start
```
